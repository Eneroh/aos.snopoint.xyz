What is aos/bns?


Ace of spades or aos was a popular free title in the early 2010's made by Ben Aksoy. The game was purchased by Jagex, however is no longer supported.

Build and shoot or bns is a player-run online, voxel FPS game based on Ace of Spades Classic.

What is openspades?


Openspades is an open-source python based client launcher created by yvt to play on build and shoot servers.

What is piqueserver?


Piqueserver is a server software based on pysnip and prioritises:

  • Authenticity
  • Stability
  • Being Mordern
  • Free-dom

It currently supports version 0.75 of Ace of Spades classic, with plans for 0.76 and is still being maintained.
